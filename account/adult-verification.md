# 성인/본인인증

## 성인인증이 필요한 경우

세이프티 필터를 OFF로 설정하려면 성인인증이 필수입니다.

{% hint style="danger" %}
미성년자 회원의 경우, 세이프티 필터 OFF가 불가능합니다.
{% endhint %}

## 성인 연령 기준

### 대한민국 국민

* **만 19세 이상**부터 성인으로 인정됩니다.
* 한국 나이 20세여도 생일이 지나지 않아 만 18세라면 인증이 불가합니다.
  * 단, 만 19세가 되는 해의 1월 1일 이후에는 예외적으로 인증 가능합니다.

### 외국인

* **만 18세 이상**부터 성인으로 인정됩니다.
* 태어난 년도 + 18년이 되는 해의 생일 자정 이후부터 인증 가능합니다.

### 해외 거주 한국 국적자

* 고객센터를 통한 별도 인증 절차가 필요합니다.

## 인증 방법

다음 상황에서 본인인증이 요청됩니다:

* **홈 화면**에서 세이프티 필터를 토글할 때
* **[마이]** 페이지에서 **민감한 콘텐츠 표시** 토글을 켤 때
* **프로필 편집**에서 성별을 변경할 때
3. **"본인인증 시작"** 버튼을 클릭합니다.
4. PortOne 본인인증 화면에서 인증을 진행합니다.
   * **PC**: 팝업 창에서 인증
   * **모바일**: 인증 페이지로 이동 후 완료 시 자동 복귀

{% hint style="info" %}
반드시 **본인 명의**의 휴대폰으로만 인증이 가능합니다.
{% endhint %}

## 인증 문제 해결

### 인증 문자를 받지 못하는 경우

* 입력한 인증 정보가 정확한지 확인해 주세요.
* 휴대폰의 스팸 차단 설정을 해제해 주세요.
* 통신사 차원의 차단 여부를 확인해 주세요:
  * SKT: 114
  * KT: 100
  * LG U+: 114 / 1544-0010

### 인증이 진행되지 않는 경우

* 브라우저의 **쿠키 사용 설정**이 허용으로 되어 있는지 확인해 주세요.
* 다른 브라우저로 시도해 보세요.
* 문제가 지속되면 고객센터로 문의해 주세요.

## 해외 거주자 인증

여권, 운전면허증 등 **이름과 생년월일을 확인할 수 있는 신분증 사본**으로 인증이 가능합니다.

{% hint style="warning" %}
신분증 제출 시 개인정보(주민번호 뒷자리, 사진 등)는 반드시 가려서 제출해 주세요.
{% endhint %}
