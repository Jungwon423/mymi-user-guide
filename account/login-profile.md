# 로그인/프로필 설정

## 로그인

1. [mymi.live](https://mymi.live)에 접속합니다.
2. 우측 상단의 **"로그인"** 버튼을 클릭합니다.
3. 가입 시 사용한 소셜 계정(Google, Apple, 카카오)으로 로그인합니다.

{% hint style="warning" %}
가입 시 사용한 것과 다른 소셜 계정으로 로그인하면 새로운 계정으로 인식됩니다.
{% endhint %}

## 프로필 설정

### 프로필 변경 방법

1. 로그인 후 **[마이]** 페이지로 이동합니다.
2. 프로필 영역을 클릭하여 **프로필 편집**으로 이동합니다.
3. **내 정보** 탭에서 다음 항목을 수정할 수 있습니다:
   * **프로필 이미지**: 원하는 이미지로 변경
   * **이름** (20자 이내): 작품 속 인물이 나를 부르는 이름
   * **닉네임** (30자 이내): 다른 사용자에게 표시되는 이름
   * **아이디**: 영문 소문자, 숫자, `_`, `-`만 사용 가능 (3~20자)
   * **생년월일**: 본인인증 후에는 수정 불가
   * **상세정보** (1000자 이내): 작품 속 인물이 나에 대해 인식할 수 있는 정보

{% hint style="info" %}
**크리에이터** 탭에서는 작품 관리, 공지사항, 댓글 관리 등 크리에이터 관련 설정을 할 수 있습니다.
{% endhint %}

### 마이 페이지 주요 메뉴

* **내 스파크**: 보유 스파크 확인 및 만료 예정 스파크 안내
* **작품 관리**: 내가 제작한 작품 관리
* **차단 관리**: 차단한 작품/크리에이터 관리
* **페르소나 설정**: 나를 대신하는 프로필 설정
* **출석 체크**: 매일 출석하면 스파크를 받을 수 있습니다
* **민감한 콘텐츠 표시**: 성인 콘텐츠 표시 여부 토글 (본인인증 필요)

{% hint style="info" %}
프로필은 다른 사용자에게 공개됩니다. 개인정보가 노출되지 않도록 주의해 주세요.
{% endhint %}
