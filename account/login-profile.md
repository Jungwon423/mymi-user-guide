# 로그인/프로필 설정

## 로그인

1. [mymi.live](https://mymi.live)에 접속합니다.
2. 우측 상단의 **"로그인"** 버튼을 클릭합니다.
3. 가입 시 사용한 소셜 계정(Google, Apple, 카카오)으로 로그인합니다.

{% hint style="warning" %}
가입 시 사용한 것과 다른 소셜 계정으로 로그인하면 새로운 계정으로 인식됩니다.
{% endhint %}

## 프로필 설정

### 프로필 변경 방법

1. 로그인 후 **\[마이]** 페이지로 이동합니다.
2. **프로필 편집** 버튼을 클릭합니다.
3. 다음 항목을 설정할 수 있습니다:
   * **프로필 이미지**: 원하는 이미지로 변경
   * **닉네임**: 다른 사용자에게 표시되는 이름
   * **자기소개**: 간단한 소개글 작성

### 닉네임 규칙

* 2~20자 이내로 설정할 수 있습니다.
* 특수문자, 공백 사용이 가능합니다.
* 부적절한 닉네임은 제재를 받을 수 있습니다.

{% hint style="info" %}
프로필은 다른 사용자에게 공개됩니다. 개인정보가 노출되지 않도록 주의해 주세요.
{% endhint %}
