# 회원가입

## 회원가입 방법

MYMI는 간편한 소셜 로그인을 지원합니다.

### 소셜 로그인으로 가입하기

1. [mymi.live](https://mymi.live)에 접속합니다.
2. **"로그인"** 버튼을 클릭합니다.
3. 원하는 소셜 계정을 선택합니다:
   * **Google** 계정으로 가입
   * **Apple** 계정으로 가입
   * **카카오** 계정으로 가입
4. 해당 소셜 서비스에서 로그인을 완료하면 자동으로 가입됩니다.

{% hint style="info" %}
소셜 로그인을 사용하면 별도의 비밀번호 설정 없이 간편하게 가입할 수 있습니다.
{% endhint %}

### 가입 시 동의 사항

회원가입 시 다음 약관에 대한 동의가 필요합니다:

* 서비스 이용약관 (필수)
* 개인정보 처리방침 (필수)
* 마케팅 수신 동의 (선택)

## 가입이 안 될 때

* 이미 같은 이메일로 가입된 계정이 있는지 확인해 주세요.
* 소셜 로그인 서비스에서 정상적으로 로그인되었는지 확인해 주세요.
* 브라우저의 팝업 차단이 설정되어 있다면 해제해 주세요.
* 문제가 지속되면 고객센터로 문의해 주세요.
