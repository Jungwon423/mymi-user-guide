# 회원가입

## 회원가입 방법

MYMI는 간편한 소셜 로그인을 지원합니다.

### 1단계: 소셜 로그인

1. [mymi.live](https://mymi.live)에 접속합니다.
2. **"로그인"** 버튼을 클릭합니다.
3. 원하는 소셜 계정을 선택합니다:
   * **Google** 계정으로 가입
   * **Apple** 계정으로 가입
   * **카카오** 계정으로 가입

{% hint style="info" %}
소셜 로그인을 사용하면 별도의 비밀번호 설정 없이 간편하게 가입할 수 있습니다.
{% endhint %}

### 2단계: 기본 정보 입력

소셜 로그인 후 신규 사용자는 다음 정보를 입력합니다:

* **이름** (20자 이내): 작품 속 인물이 나를 부르는 이름입니다.
* **성별**: 여성 / 남성 / 기타 중 선택합니다.
* **아이디** (3~20자): 영문 소문자, 숫자, `_`, `-`만 사용 가능합니다. 중복 확인이 자동으로 진행됩니다.
* **닉네임** (30자 이내): 다른 사용자에게 표시되는 이름입니다.
* **생년월일**: 연도, 월, 일을 입력합니다.

### 3단계: 약관 동의

회원가입 시 다음 약관에 대한 동의가 필요합니다:

* 이용약관 동의 (필수)
* 개인정보 수집 및 이용 동의 (필수)
* 서비스 유의사항 (필수)
* 마케팅 정보 수신 동의 (선택)

## 가입이 안 될 때

* 이미 같은 이메일로 가입된 계정이 있는지 확인해 주세요.
* 소셜 로그인 서비스에서 정상적으로 로그인되었는지 확인해 주세요.
* 브라우저의 팝업 차단이 설정되어 있다면 해제해 주세요.
* 문제가 지속되면 고객센터로 문의해 주세요.
