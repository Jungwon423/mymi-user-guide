# 계정 탈퇴

## 탈퇴 방법

1. **\[마이]** 페이지로 이동합니다.
2. **\[설정]** → **\[계정 관리]**를 선택합니다.
3. 하단의 **"계정 탈퇴"** 버튼을 클릭합니다.
4. 안내 사항을 확인한 후 탈퇴를 진행합니다.

## 탈퇴 전 유의사항

{% hint style="danger" %}
계정 탈퇴 시 아래 내용은 **복구할 수 없습니다**.
{% endhint %}

* 모든 대화 기록이 삭제됩니다.
* 제작한 캐릭터가 삭제됩니다.
* 구매한 아이템 및 포인트가 소멸됩니다.
* 동일한 소셜 계정으로 재가입하더라도 이전 데이터는 복구되지 않습니다.

## 미사용 구독/결제가 있는 경우

* 탈퇴 전 진행 중인 구독을 반드시 해지해 주세요.
* 환불 가능한 결제 건이 있다면 먼저 환불을 진행해 주세요.
* 탈퇴 후에는 결제 관련 처리가 불가합니다.

{% hint style="info" %}
탈퇴를 고민 중이시라면, 일시적으로 로그아웃 상태로 유지하는 것도 방법입니다. 계정은 그대로 보존됩니다.
{% endhint %}
