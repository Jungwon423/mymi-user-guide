# 계정 탈퇴

## 탈퇴 방법

1. **[마이]** 페이지로 이동합니다.
2. 프로필 영역을 클릭하여 **프로필 편집**으로 이동합니다.
3. 하단의 **"계정 비활성화"** 버튼을 클릭합니다.
4. 확인 다이얼로그에서 **"삭제"**를 클릭하면 탈퇴가 완료됩니다.

## 탈퇴 전 유의사항

{% hint style="danger" %}
계정 탈퇴 시 아래 내용은 **복구할 수 없습니다**.
{% endhint %}

* 모든 대화 기록이 삭제됩니다.
* 제작한 작품이 삭제됩니다.
* 보유한 스파크 및 크레딧이 소멸됩니다.
* 동일한 소셜 계정으로 재가입하더라도 이전 데이터는 복구되지 않습니다.

{% hint style="warning" %}
탈퇴 후 **30일 동안 동일한 계정으로 재가입이 불가**합니다.
{% endhint %}

## 미사용 결제가 있는 경우

* 환불 가능한 결제 건이 있다면 먼저 환불을 진행해 주세요.
* 탈퇴 후에는 결제 관련 처리가 불가합니다.

{% hint style="info" %}
탈퇴를 고민 중이시라면, 일시적으로 로그아웃 상태로 유지하는 것도 방법입니다. 계정은 그대로 보존됩니다.
{% endhint %}
