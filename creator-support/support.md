# 크리에이터 응원하기

## 크리에이터 프로필

크리에이터 프로필 페이지에서 제작자의 작품 목록, 공지사항, 댓글을 확인할 수 있습니다.

### 크리에이터 프로필로 이동하기

1. 작품 상세 페이지에서 크리에이터 이름을 클릭합니다.
2. 크리에이터 프로필 페이지로 이동합니다.

## 팔로우

크리에이터를 **팔로우**하면 해당 크리에이터의 새 작품 소식을 받을 수 있습니다.

* 프로필 페이지에서 **"팔로우"** 버튼을 클릭합니다.
* 팔로우를 해제하려면 다시 버튼을 클릭합니다.

## 댓글

크리에이터 프로필에 댓글을 남겨 응원의 메시지를 전달할 수 있습니다.

* 댓글 작성, 수정, 삭제가 가능합니다.
* **비밀 댓글**로 작성하면 작성자와 크리에이터만 확인할 수 있습니다.
* 댓글에 좋아요를 남기거나 답글을 달 수 있습니다.

## 크리에이터 대시보드

크리에이터는 **대시보드**에서 자신의 작품 성과를 확인할 수 있습니다.

* **KPI**: 총 메시지 수, 내 메시지 수, 획득 리프
* **기간별 추이**: 일별 메시지/리프 데이터 차트
* **TOP 콘텐츠**: 주간/월간 인기 작품
* **콘텐츠별 성과**: 각 작품의 상세 성과 확인

{% hint style="info" %}
크리에이터의 작품이 사용될수록 **리프**(제작자 보상 크레딧)가 적립됩니다. 리프는 스파크나 파이어로 전환할 수 있습니다.
{% endhint %}
