# 응원하기

## 응원하기란?

응원하기는 마음에 드는 캐릭터의 제작자에게 **감사와 지지를 표현**할 수 있는 기능입니다. 응원을 통해 제작자는 더 좋은 캐릭터를 만들 수 있는 동기부여를 받습니다.

## 응원하는 방법

1. 캐릭터 상세 페이지로 이동합니다.
2. **"응원하기"** 버튼을 클릭합니다.
3. 응원 금액을 선택합니다.
4. 결제를 완료하면 응원이 전달됩니다.

## 응원 시 참고사항

* 응원 금액은 제작자에게 전달됩니다.
* 응원과 함께 짧은 메시지를 남길 수 있습니다.
* 응원 내역은 [마이] 페이지에서 확인할 수 있습니다.

{% hint style="info" %}
응원은 자발적인 후원 기능이며, 캐릭터 이용에 필수가 아닙니다.
{% endhint %}
