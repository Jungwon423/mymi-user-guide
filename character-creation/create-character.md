# 나만의 AI 캐릭터 제작하기

## 캐릭터 제작 시작

1. 하단 메뉴에서 **"제작"** 탭을 선택합니다.
2. **"새 캐릭터 만들기"** 버튼을 클릭합니다.

## 기본 정보 설정

### 필수 항목

* **캐릭터 이름**: 캐릭터의 이름을 입력합니다.
* **캐릭터 이미지**: 캐릭터를 대표하는 이미지를 업로드합니다.
* **캐릭터 설명**: 다른 사용자에게 보이는 캐릭터 소개글을 작성합니다.
* **태그**: 캐릭터의 장르, 성격 등을 태그로 설정합니다.

### 선택 항목

* **카테고리**: 캐릭터가 속할 카테고리를 선택합니다.
* **공개/비공개**: 다른 사용자에게 캐릭터를 공개할지 설정합니다.

## 상세 설정

### 캐릭터 프롬프트

캐릭터의 **성격, 말투, 배경 설정** 등을 작성합니다. 이 내용이 AI가 캐릭터를 연기하는 기본 지침이 됩니다.

{% hint style="success" %}
**좋은 프롬프트 작성 팁**

* 구체적이고 명확하게 작성하세요.
* 캐릭터의 행동 지침과 배경 정보를 분리하세요.
* 마크다운이나 YAML 형식으로 구조화하면 더 좋습니다.
{% endhint %}

### 첫 인사 메시지

캐릭터가 대화를 시작할 때 보내는 첫 메시지를 설정합니다.

* 캐릭터의 성격이 잘 드러나도록 작성하세요.
* 대화의 시작점이 되는 상황을 설정할 수 있습니다.

### 로어북

캐릭터의 세계관, 인물 관계 등 보충 설정을 로어북으로 관리할 수 있습니다.

➡️ 자세한 내용은 [로어북](lorebook.md) 페이지를 참고해 주세요.

## 캐릭터 공개

1. 모든 설정을 완료한 후 **"저장"** 버튼을 클릭합니다.
2. **공개** 설정을 활성화하면 다른 사용자들이 캐릭터와 대화할 수 있습니다.
3. 공개된 캐릭터는 메인 페이지에서 탐색 가능합니다.

{% hint style="warning" %}
캐릭터 제작 시 [MYMI 정책 및 이용약관](../policy/terms.md)을 반드시 준수해 주세요. 정책을 위반하는 캐릭터는 삭제될 수 있습니다.
{% endhint %}
