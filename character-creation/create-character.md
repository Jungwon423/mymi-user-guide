# 나만의 AI 작품 제작하기

MYMI의 작품 제작은 **7단계**로 진행됩니다. 각 단계를 탭으로 자유롭게 이동하면서 작업할 수 있습니다.

## Step 1: 기본 정보

작품의 핵심 정보를 입력합니다.

* **작품 이름** (필수, 50자): 작품의 제목
* **한 줄 소개** (필수, 40자): 작품의 핵심 장르/분위기 표현
* **세계관** (필수): 작품의 배경, 설정, 시간대 등
* **등장인물** (필수): 이름(50자)과 설명을 입력합니다. 여러 명 추가 가능
* **비밀** (선택): 작품의 숨겨진 설정이나 중요한 메모

{% hint style="info" %}
세계관 + 등장인물 설명 + 비밀의 합계가 **최대 10,000자**입니다. 진행바로 사용량을 확인할 수 있습니다.
{% endhint %}

## Step 2: 에셋

대화 중 사용할 이미지를 관리합니다.

* **직접 업로드**: 이미지 파일을 업로드합니다.
* **AI 이미지 생성**: AI로 일관성 있는 이미지를 생성합니다.

각 에셋에는 **상황 설명** (필수, 120자)을 입력해야 합니다. AI가 언제 이 이미지를 사용할지 판단하는 기준이 됩니다.

➡️ AI 이미지 생성에 대한 자세한 내용은 [이미지 출력 방식 안내](image-output.md)를 참고하세요.

## Step 3: 도입부

작품의 첫 메시지를 설정합니다. 여러 도입부를 만들어 다양한 시작 시나리오를 제공할 수 있습니다.

* **제목** (선택): 도입부 구분용 이름
* **메시지 내용** (필수, 2,000자): 첫 인사 및 상황 설정
* 에셋에서 추가한 이미지를 `img:[slug]` 형식으로 삽입할 수 있습니다.
* **미리보기** 버튼으로 실제 채팅 화면을 확인할 수 있습니다.

## Step 4: 기타 설정

### 해시태그 (필수)

* **타겟 독자층** (필수): 남성향 / 여성향 중 선택
* **기타 태그**: 관계, 장르, 설정, 성향, 메타, 종족 등 카테고리별 다중 선택
* **AI 추천받기** 버튼으로 자동 추천도 가능합니다.

### 공개 설정

* **공개 / 비공개**: 다른 사용자에게 작품을 공개할지 설정
* **오리지널**: MYMI 독점 공개 시 더 높은 리프 인센티브 (3개월 내 해제 불가)
* **이미지 출처**: MYMI 생성 / 외부 제작 중 선택
* **콘텐츠 등급**: 성인용 / 전체 이용가 중 선택

## Step 5: 로어북

특정 키워드가 대화에 등장하면 관련 설정이 자동으로 AI에 주입됩니다.

➡️ 자세한 내용은 [로어북](lorebook.md) 페이지를 참고하세요.

## Step 6: 컴포넌트

대화 중 시각적인 UI 요소를 제공하는 커스텀 컴포넌트를 정의합니다. 스탯 표시, 인벤토리, 선택지 등을 만들 수 있습니다.

* **용도 설명** (선택, 500자): AI가 컴포넌트를 언제 사용할지 참고
* **JSX 코드** (필수, 10,000자): 함수형 컴포넌트 형식으로 작성
* 미리보기로 렌더링 결과를 확인할 수 있습니다.

## Step 7: 선택사항

### 프로필 디자인 (HTML)

HTML로 작품 프로필 페이지를 직접 꾸밀 수 있습니다 (최대 50,000자). 안전성 검증이 자동으로 진행됩니다.

### 추천 페르소나

사용자에게 제안할 역할을 미리 설정합니다. 닉네임, 생년월일, 성별, 상세정보를 입력합니다.

### 제작자의 한 마디

작품 소개, 제작 의도, 팁 등을 자유롭게 작성합니다 (최대 3,000자).

{% hint style="warning" %}
작품 제작 시 [MYMI 정책 및 이용약관](../policy/terms.md)을 반드시 준수해 주세요. 정책을 위반하는 작품은 삭제될 수 있습니다.
{% endhint %}
