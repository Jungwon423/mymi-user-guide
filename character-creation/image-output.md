# 이미지 출력 방식 안내

## 이미지 출력 방식

MYMI에서 캐릭터가 대화 중 이미지를 출력하는 방식은 두 가지가 있습니다.

### 1. 내부 이미지 (플랫폼 업로드)

* 제작자가 직접 MYMI에 업로드한 이미지를 사용합니다.
* `img:[slug]` 형식으로 프롬프트에서 호출합니다.
* 안정적이고 빠른 로딩이 장점입니다.

### 2. 외부 이미지 (URL)

* 외부 이미지 호스팅 서비스의 URL을 사용합니다.
* 프롬프트에서 이미지 URL을 직접 참조합니다.

{% hint style="danger" %}
**중요**: 내부 이미지와 외부 이미지를 **혼용하면 안 됩니다**. 혼용 시 이미지 출력 실패 또는 오류가 발생할 수 있습니다. 하나의 방식만 선택하여 사용해 주세요.
{% endhint %}

## 이미지 출력 설정 시 주의사항

* 실제로 사용할 이미지만 유지하세요. 사용하지 않는 이미지는 제거해 주세요.
* `img:[slug]` 방식을 사용할 때, 키워드 방식은 사용하지 마세요.
* 이미지 파일 크기가 너무 크면 로딩이 느려질 수 있습니다.

## 이미지 업로드 방법

1. 캐릭터 편집 화면에서 **"이미지"** 탭을 선택합니다.
2. **"이미지 추가"** 버튼을 클릭합니다.
3. 이미지 파일을 선택하여 업로드합니다.
4. 각 이미지에 **slug**(고유 식별자)를 지정합니다.
5. 프롬프트에서 `img:[slug]`로 해당 이미지를 호출합니다.

{% hint style="info" %}
이미지 출력 관련 기능은 지속적으로 개선되고 있습니다. 최신 업데이트 내용을 확인해 주세요.
{% endhint %}
