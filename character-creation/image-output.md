# 이미지 출력 방식 안내

## 이미지 추가 방법

작품 제작 **에셋 단계(Step 2)**에서 이미지를 추가할 수 있습니다.

### 1. 직접 업로드

* 이미지 파일을 직접 업로드합니다.
* 첫 번째 업로드 이미지가 자동으로 프로필 이미지로 설정됩니다.

### 2. AI 이미지 생성

**"AI 이미지"** 버튼을 클릭하면 AI로 일관성 있는 이미지를 생성할 수 있습니다.

#### 기본 이미지 생성

캐릭터의 기본 모습을 생성합니다.

1. **이미지 유형 선택**: 캐릭터 / 배경·사물
2. **캐릭터 할당**: 기본 정보에서 등록한 등장인물 중 선택
3. **성별 선택**: 여성 / 남성
4. **AI 모델 선택**: 다양한 이미지 생성 스타일 중 선택
5. **아티스트 스타일 선택**: 선택한 모델이 제공하는 스타일
6. **태그 선택**: 머리, 눈, 얼굴, 피부, 신체 등 카테고리별 태그 선택 (머리, 눈은 색상 지정 가능)
7. **확인 및 생성**: 선택 요약 확인 후 생성

#### 상황 이미지 생성

기본 이미지를 기반으로 다양한 표정/동작의 변형 이미지를 생성합니다.

1. **표정 선택**: 미소, 화남, 슬픔, 울음 등 56가지 표정
2. **동작 선택**: 서기, 앉기, 가리키기 등 180가지 이상의 동작
3. **선택사항**: 배경, 의상, 소품 등 추가 설정
4. **확인 및 생성**

{% hint style="info" %}
이미지 생성에는 **스파크 또는 파이어** 크레딧이 소비됩니다.
{% endhint %}

## 이미지 사용 방법

* 각 이미지에는 `img:[slug]` 형식의 **고유 식별자**가 자동 생성됩니다.
* 도입부(Step 3)에서 이 slug를 사용하여 이미지를 삽입할 수 있습니다.
* 각 에셋에는 **상황 설명** (필수, 120자)을 입력해야 합니다. AI가 대화 중 언제 이 이미지를 표시할지 판단하는 기준입니다.

{% hint style="danger" %}
**중요**: 내부 이미지(플랫폼 업로드/AI 생성)와 외부 이미지(URL)를 **혼용하면 안 됩니다**. 하나의 방식만 선택하여 사용해 주세요.
{% endhint %}

## 프로필 이미지 설정

* 에셋 목록에서 별 아이콘을 클릭하여 **프로필 이미지**를 지정합니다.
* 프로필 이미지는 작품 탐색 시 대표로 표시됩니다.
* 한 번에 1개만 선택 가능합니다.
