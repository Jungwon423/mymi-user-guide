# 로어북

## 로어북이란?

로어북은 대화의 맥락에 깊이를 더하는 **설정집**입니다. 채팅 중 사전 설정된 키워드가 언급되면, 해당 정보가 자동으로 AI 프롬프트에 추가되어 작품 속 인물이 세계관, 인물 관계, 아이템 정보, 과거 사건 등을 자연스럽게 반영할 수 있게 합니다.

{% hint style="info" %}
**핵심 구분**: 작품의 핵심 설정은 **기본 정보(세계관/등장인물)**에, 보충 정보는 **로어북**에 작성합니다.
{% endhint %}

## 로어북 항목 추가

1. 작품 제작 화면에서 **"로어북"** 탭(Step 5)을 선택합니다.
2. **"엔트리 추가"** 버튼을 클릭합니다.
3. 다음 정보를 입력합니다:
   * **엔트리 이름** (필수, 50자): 항목의 제목
   * **트리거 키워드** (최대 5개, 각 10자): 이 항목이 활성화되는 단어. Enter 또는 쉼표로 추가
   * **내용** (필수, 400자): 키워드가 언급될 때 AI에게 제공되는 정보

{% hint style="info" %}
로어북은 **최대 100개 항목**까지 등록할 수 있습니다.
{% endhint %}

## 키워드 설정 팁

### 피해야 할 키워드

* "회사", "학교", "집" 같은 **일반적인 단어**
* 너무 일반적인 키워드는 불필요하게 자주 활성화됩니다.

### 추천하는 키워드

* "드래곤의 둥지", "비밀 조직" 같은 **구체적이고 고유한 표현**
* 필요한 순간에만 정확히 활성화됩니다.

## 활용 사례

* **세계관 설정**: 판타지 세계의 지리, 역사, 마법 체계 등
* **인물 관계**: 등장인물과 관련된 다른 인물들의 정보
* **아이템/장소**: 특수한 아이템이나 중요 장소에 대한 설명
* **과거 사건**: 스토리 진행에 필요한 과거 이벤트 정보

➡️ 더 자세한 활용법은 [로어북 가이드라인 [심화]](../creation-guide/lorebook-guide.md)를 참고하세요.
