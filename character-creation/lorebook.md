# 로어북

## 로어북이란?

로어북은 대화의 맥락에 깊이를 더하는 **설정집**입니다. 채팅 중 사전 설정된 키워드가 언급되면, 해당 정보가 자동으로 AI 프롬프트에 추가되어 캐릭터가 세계관, 인간관계, 아이템 정보, 과거 사건 등을 자연스럽게 반영할 수 있게 합니다.

{% hint style="info" %}
**핵심 구분**: 캐릭터의 핵심 설정은 **상세 설정**에, 보충 정보와 세계관은 **로어북**에 작성합니다.
{% endhint %}

## 로어북 항목 추가

1. 캐릭터 편집 화면에서 **"로어북"** 탭을 선택합니다.
2. **"새 항목 추가"** 버튼을 클릭합니다.
3. 다음 정보를 입력합니다:
   * **키워드**: 이 항목이 활성화되는 트리거 단어
   * **내용**: 키워드가 언급될 때 AI에게 제공되는 정보
   * **우선순위**: 여러 항목이 동시에 활성화될 때의 순서

## 키워드 설정 팁

### 피해야 할 키워드

* ❌ "회사", "학교", "집" 같은 **일반적인 단어**
* 너무 일반적인 키워드는 불필요하게 자주 활성화됩니다.

### 추천하는 키워드

* ✅ "드래곤의 둥지", "비밀 조직" 같은 **구체적이고 고유한 표현**
* 필요한 순간에만 정확히 활성화됩니다.

## 우선순위 관리

{% hint style="warning" %}
로어북은 한 대화에서 **최대 5개까지만 동시 활성화**됩니다. 6개 이상의 키워드가 포함되면 상위 5개만 반영되고 나머지는 누락됩니다.
{% endhint %}

* 캐릭터의 정체성과 핵심 행동 원리에 해당하는 정보를 **목록 상단**에 배치하세요.
* 부가적인 세계관 정보는 하단에 배치하세요.

## 활용 사례

* **세계관 설정**: 판타지 세계의 지리, 역사, 마법 체계 등
* **인물 관계**: 캐릭터와 관련된 다른 인물들의 정보
* **아이템/장소**: 특수한 아이템이나 중요 장소에 대한 설명
* **과거 사건**: 스토리 진행에 필요한 과거 이벤트 정보

➡️ 더 자세한 활용법은 [로어북 가이드라인 [심화]](../creation-guide/lorebook-guide.md)를 참고하세요.
