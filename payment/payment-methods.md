# 결제수단

## 크레딧 시스템

MYMI에서는 3가지 크레딧을 사용합니다.

* **스파크 (Spark)**: 메시지 전송에 사용되는 프리미엄 크레딧. AI 모델에 따라 소비량이 다릅니다.
* **리프 (Leaf)**: 크리에이터 보상 크레딧. 작품이 사용될수록 적립됩니다.
* **파이어 (Fire)**: 이미지 생성에 사용되는 크레딧. 리프에서 전환할 수 있습니다.

## 스파크 패키지

| 가격 | 스파크 | 비고 |
| --- | --- | --- |
| **2,500원** | 5,000 | 초회 1회 한정 (본인인증 필요) |
| **5,000원** | 5,000 | |
| **10,000원** | 10,000 | |
| **30,000원** | 31,500 | 5% 보너스 |
| **50,000원** | 55,000 | 10% 보너스 |
| **100,000원** | 115,000 | 15% 보너스 |

## 무료 스파크 획득 방법

* **회원가입**: 가입 시 기본 스파크 지급
* **출석 체크**: 매일 출석 시 200 스파크 지급 (연속 출석 시 최대 2배)

## 결제 수단

카드 결제를 지원합니다 (PortOne 결제).

{% hint style="warning" %}
**하나SK카드**는 사용이 불가하며, **일시불만** 지원됩니다.
{% endhint %}

## 스파크 이용 안내

* 온라인 상품 구매 후 즉시 사용 가능합니다.
* 사용하는 AI 모델에 따라 스파크가 차감됩니다. 모델에 따라 5~300 스파크까지 차감될 수 있으며, 보다 고급 모델은 더 많은 스파크가 소비됩니다.
* 스파크가 없을 경우 대화를 이어갈 수 없습니다.
* 구매한 상품은 결제일로부터 **1년 이내**에 사용할 수 있습니다.

## 결제 시 주의사항

* 결제 전 결제 금액과 상품 내용을 반드시 확인해 주세요.
* 법정대리인의 동의 없이 미성년자의 결제는 취소될 수 있습니다.
* 결제 오류가 발생하면 잠시 후 다시 시도하거나 고객센터로 연락해 주세요.
