# 결제취소 및 환불

## 환불 정책

### 환불 가능한 경우

* 결제 후 서비스를 이용하지 않은 경우
* 결제 오류로 중복 결제된 경우
* 서비스 장애로 정상적인 이용이 불가했던 경우

### 환불이 불가능한 경우

* 이미 서비스를 이용한 경우 (메시지 전송, 콘텐츠 이용 등)
* 구독 기간이 경과한 경우
* 이벤트/프로모션으로 제공된 무료 아이템

## 환불 신청 방법

1. **[마이]** 페이지로 이동합니다.
2. **[결제 내역]**을 선택합니다.
3. 환불을 원하는 결제 건을 선택합니다.
4. **"환불 신청"** 버튼을 클릭합니다.
5. 환불 사유를 입력하고 신청을 완료합니다.

## 환불 처리 기간

| 결제수단 | 처리 기간 |
| --- | --- |
| 신용/체크카드 | 3~7 영업일 |
| 간편결제 | 1~3 영업일 |
| 휴대폰 결제 | 당월 취소만 가능 |

{% hint style="warning" %}
환불 처리 기간은 결제사 및 카드사 사정에 따라 변동될 수 있습니다.
{% endhint %}

## 구독 해지

1. **[마이]** → **[구독 관리]**로 이동합니다.
2. **"구독 해지"** 버튼을 클릭합니다.
3. 해지 후에도 현재 구독 기간이 만료될 때까지 서비스를 이용할 수 있습니다.

{% hint style="info" %}
구독 해지와 환불은 별개의 절차입니다. 환불이 필요한 경우 별도로 환불 신청을 해주세요.
{% endhint %}
