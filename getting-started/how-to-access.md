# MYMI 접속 방법

## 웹 브라우저로 접속

MYMI는 웹 브라우저를 통해 접속할 수 있습니다.

1. 웹 브라우저(Chrome, Safari, Edge 등)를 엽니다.
2. 주소창에 [**mymi.live**](https://mymi.live)를 입력합니다.
3. MYMI 메인 페이지가 표시됩니다.

{% hint style="success" %}
**권장 브라우저**: Chrome, Safari, Edge 최신 버전을 권장합니다.
{% endhint %}

## 모바일 접속

### iOS (아이폰/아이패드)

1. Safari 또는 Chrome 브라우저에서 [mymi.live](https://mymi.live)에 접속합니다.
2. 하단의 공유 버튼을 탭합니다.
3. **"홈 화면에 추가"**를 선택하면 앱처럼 사용할 수 있습니다.

### Android

1. Chrome 브라우저에서 [mymi.live](https://mymi.live)에 접속합니다.
2. 우측 상단 메뉴(⋮)를 탭합니다.
3. **"홈 화면에 추가"**를 선택하면 앱처럼 사용할 수 있습니다.

{% hint style="warning" %}
인앱 브라우저(카카오톡, 인스타그램 등에서 링크를 열 때)에서는 일부 기능이 정상 작동하지 않을 수 있습니다. 외부 브라우저로 열어주세요.
{% endhint %}

## 접속이 안 될 때

* 인터넷 연결 상태를 확인해 주세요.
* 브라우저 캐시를 삭제한 후 다시 시도해 주세요.
* 다른 브라우저로 접속을 시도해 보세요.
* 문제가 지속되면 고객센터로 문의해 주세요.
