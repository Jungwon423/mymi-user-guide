# MOBILE/PC 웹사이트 쿠키 삭제

인증이 진행되지 않거나 로그인에 문제가 있을 때, 브라우저의 쿠키를 삭제하면 해결되는 경우가 많습니다.

## PC 브라우저

### Chrome

1. 우측 상단 메뉴(⋮) → **설정** 클릭
2. **개인 정보 보호 및 보안** → **인터넷 사용 기록 삭제**
3. **쿠키 및 기타 사이트 데이터** 체크
4. **인터넷 사용 기록 삭제** 클릭

### Safari

1. 상단 메뉴 **Safari** → **환경설정**
2. **개인 정보 보호** 탭 선택
3. **웹 사이트 데이터 관리** 클릭
4. **모두 제거** 클릭

### Edge

1. 우측 상단 메뉴(⋯) → **설정** 클릭
2. **개인 정보, 검색 및 서비스** 선택
3. **지금 검색 데이터 지우기** → **지울 항목 선택**
4. **쿠키 및 기타 사이트 데이터** 체크 후 삭제

## 모바일 브라우저

### iOS Safari

1. **설정** → **Safari** 선택
2. **방문 기록 및 웹 사이트 데이터 지우기** 탭
3. **방문 기록 및 데이터 지우기** 확인

### Android Chrome

1. Chrome 앱에서 우측 상단 메뉴(⋮) 탭
2. **설정** → **개인 정보 보호 및 보안**
3. **인터넷 사용 기록 삭제** 탭
4. **쿠키 및 사이트 데이터** 체크 후 삭제

{% hint style="warning" %}
쿠키를 삭제하면 MYMI를 포함한 다른 웹사이트에서도 로그아웃될 수 있습니다.
{% endhint %}

{% hint style="info" %}
쿠키 삭제 후에도 문제가 해결되지 않으면 고객센터로 문의해 주세요.
{% endhint %}
