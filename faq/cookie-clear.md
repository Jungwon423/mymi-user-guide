# MOBILE/PC 웹사이트 쿠키 삭제

서비스 이용 시 마이프로필, 로그아웃 버튼이 보이지 않거나 인증이 진행되지 않는다면 아래 방법을 시도해 주세요.

## 세션 정보 초기화

먼저 앱 내에서 아래 경로로 세션 정보를 초기화해 보세요.

**[마이페이지] → [설정] → [세션정보 초기화] → [세션정보 초기화하기]**

위 방법으로 해결되지 않는 경우, 아래 브라우저별 쿠키 삭제를 진행해 주세요.

---

## Android

### Chrome 브라우저

1. 주소창 왼쪽 **설정 버튼**을 클릭합니다.
2. **쿠키 및 사이트 데이터**를 관리합니다.
3. MYMI 관련 쿠키를 삭제합니다.

### Samsung Internet 브라우저

1. 주소창 왼쪽 **자물쇠 버튼**을 클릭합니다.
2. **쿠키 00개 사용 중** 항목을 확인합니다.
3. 옆의 **휴지통 버튼**을 클릭하여 삭제합니다.

---

## iOS

### Safari 브라우저

1. **[설정]** → **[앱]** → **[Safari]** → **[고급]** → **[웹사이트 데이터]**로 이동합니다.
2. **[편집]**을 눌러 mymi.live 관련 데이터를 삭제합니다.

### Chrome 브라우저

1. 하단의 **... 버튼**을 탭합니다.
2. **[인터넷 사용 기록 삭제]**를 선택합니다.
3. **[쿠키, 사이트 데이터]**를 체크합니다.
4. **[인터넷 사용 기록 삭제]**를 클릭합니다.

---

## PC

### Chrome

1. 주소창 왼쪽 **설정 버튼**을 클릭합니다.
2. **[쿠키 및 사이트 데이터]** → **[기기 내 사이트 데이터 관리]**를 선택합니다.
3. **방문 중인 사이트의 데이터**를 모두 삭제합니다.

### Edge

1. 주소창 왼쪽 **자물쇠 버튼**을 클릭합니다.
2. **[쿠키 및 사이트 데이터]** → **[쿠키]** → **[허용됨]**에서 mymi.live 관련 항목을 선택 후 제거합니다.

### Whale

1. 주소창 왼쪽 **자물쇠 버튼**을 클릭합니다.
2. **[쿠키 및 사이트 데이터]** → **[기기 내 사이트 데이터 관리]**를 선택합니다.
3. **방문 중인 사이트의 데이터**를 모두 삭제합니다.

{% hint style="warning" %}
쿠키를 삭제하면 MYMI를 포함한 다른 웹사이트에서도 로그아웃될 수 있습니다.
{% endhint %}

{% hint style="info" %}
위 방법으로도 문제가 해결되지 않으면 고객센터로 문의해 주세요.
{% endhint %}
