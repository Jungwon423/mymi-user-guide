# 로어북 가이드라인 [심화]

{% hint style="info" %}
본 가이드는 로어북의 기본 사용법을 알고 있는 제작자를 대상으로 합니다. 먼저 [로어북](../character-creation/lorebook.md) 페이지를 읽어주세요.
{% endhint %}

## 효과적인 로어북 작성의 핵심 원칙

### 1. 키워드 선정

로어북의 핵심은 **적절한 키워드 선정**입니다.

| ❌ 피해야 할 키워드 | ✅ 추천하는 키워드 |
| --- | --- |
| "회사", "대표", "커피" | "MYMI 대표", "탈출 계획" |
| 일반적 → 불필요한 호출 증가 | 구체적 → 필요한 순간에만 활성화 |

### 2. 엔트리 관리

로어북은 **최대 100개 항목**까지 등록할 수 있으며, 항목당 **키워드 5개** (각 10자), **내용 400자**까지 작성 가능합니다.

{% hint style="info" %}
핵심 설정은 **기본 정보(세계관/등장인물)**에 작성하고, 보충 정보는 **로어북**에 작성하세요.
{% endhint %}

### 3. 포맷팅

마크다운 구조화를 통해 정보의 위계를 명확히 합니다.

{% hint style="success" %}
로어북 내용은 **헤더 3단계(###)**부터 시작하는 것을 권장합니다. 이렇게 하면 모델이 이를 메인 설정이 아닌 종속 정보로 인식합니다.
{% endhint %}

**예시:**

```markdown
### 비밀 조직 "그림자의 손"
- 설립: 100년 전, 대마법사 아르카누스가 창설
- 목적: 금지된 마법의 연구와 보존
- 구성원: 7명의 핵심 멤버로 구성
  - 리더: 현재 {{char}}의 스승
  - 실행대장: 정체불명의 가면 전사
```

## 실전 활용 패턴

### 세계관 로어북

```markdown
### 아르카디아 왕국
- 위치: 대륙 중앙부의 평원 지대
- 정치 체제: 입헌군주제
- 특징: 마법과 과학이 공존하는 사회
- 현재 상황: 북부 국경에서 마족의 침입이 잦아지고 있음
```

### 인물 관계 로어북

```markdown
### 서연 ({{char}}의 소꿉친구)
- 관계: 어린 시절부터 함께 자란 친구
- 성격: 밝고 활발, {{char}}와 정반대
- 비밀: {{char}}에게 고백하지 못한 감정이 있음
- {{char}}의 태도: 소중한 친구로 여기지만, 감정은 눈치채지 못함
```

### 아이템 로어북

```markdown
### 별의 목걸이
- 종류: 마법 아티팩트
- 능력: 착용자의 마력을 2배로 증폭
- 부작용: 장시간 사용 시 정신적 피로 누적
- 현재: {{char}}가 스승에게 물려받아 소지 중
```
