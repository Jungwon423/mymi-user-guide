# 채팅방 설정하기

채팅방 우측 하단의 **설정 메뉴(⋮)**를 클릭하면 다양한 설정을 할 수 있습니다.

## 대화 관리

* **새 대화 시작**: 같은 작품에서 새로운 채팅방을 만듭니다.
* **이전 대화 목록**: 이 작품에서 나눈 이전 대화들을 확인합니다.
* **채팅방 ID 복사**: 문의 시 사용할 채팅방 ID를 복사합니다.

## 대화 설정

### 문체 선택

* **소설**: 서술적인 문체로 대화합니다.
* **라이트노벨**: 가볍고 경쾌한 문체로 대화합니다.

### 토글 옵션

* **스토리 부스터**: 스토리 전개를 더 적극적으로 진행합니다.
* **사칭 허용**: AI가 사용자의 행동이나 대사를 묘사할 수 있도록 허용합니다.
* **한 장면씩**: 한 번에 한 장면씩만 진행합니다.
* **메모리 사용**: 장기기억 및 기억 조각을 대화에 활용합니다.

## 정보 관리

* **내 정보 (페르소나)**: 채팅방에서 사용할 페르소나를 변경하거나 수정합니다.
* **유저 노트**: 작품 속 인물에게 전달할 "주인공 설정"을 자유롭게 메모합니다.
* **대화 요약**: AI가 자동으로 생성한 대화 요약을 확인합니다.
* **메모리**: 장기기억과 기억 조각 목록을 확인하고 검색합니다.

## UI 커스터마이징

헤더의 **팔레트 아이콘**을 클릭하면 UI 설정 패널이 열립니다.

* **채팅 모드 전환**: 채팅 / 1:1 / 노벨 모드 선택
* **글자 크기 및 줄 간격**: +/- 버튼으로 조절
* **색상 설정**: 캐릭터 대사색, 사용자 대사색, 서술색 등을 각각 설정
* **글꼴**: 고딕, 나눔고딕, 나눔명조 등 선택
* **스트리밍 속도**: 메시지가 표시되는 속도 조절
* **캐릭터 이름 표시**: ON/OFF
* **이미지 높이** (노벨 모드): 20~80vh 범위 조절
* **비디오 루프** (노벨/1:1 모드): 비디오 반복 재생 설정

## 북마크 & 분기

### 북마크

메시지에 호버하면 나타나는 **북마크 아이콘**을 클릭하여 중요한 지점을 저장할 수 있습니다. 라벨을 입력하여 구분하면 편리합니다.

### 대화 분기

북마크 지점에서 **분기 아이콘**을 클릭하면 해당 시점부터 새로운 대화 라인을 시작할 수 있습니다. 다른 선택지로 이야기를 진행해보고 싶을 때 유용합니다.

## 캡처 모드

설정 메뉴에서 **"캡처"**를 선택하면 캡처 모드로 진입합니다.

1. 캡처할 메시지를 클릭하여 선택합니다.
2. **"내 대사 제외"** 토글로 사용자 메시지를 제외할 수 있습니다.
3. **"이미지 저장"** 버튼을 클릭하면 스크린샷이 다운로드됩니다.

{% hint style="info" %}
채팅방 설정은 해당 채팅방에만 적용되며, 다른 채팅방에는 영향을 주지 않습니다.
{% endhint %}
