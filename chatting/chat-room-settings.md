# 채팅방 설정하기

## 채팅방 관리

### 새 채팅방 만들기

같은 캐릭터와 여러 개의 채팅방을 만들 수 있습니다.

1. 캐릭터 상세 페이지에서 **"새 채팅방"** 버튼을 클릭합니다.
2. 새로운 채팅방이 생성되며 캐릭터의 첫 인사가 표시됩니다.

### 채팅방 삭제

1. 채팅방 목록에서 삭제할 채팅방을 선택합니다.
2. **설정** 아이콘을 클릭합니다.
3. **"채팅방 삭제"**를 선택합니다.

{% hint style="danger" %}
삭제된 채팅방은 복구할 수 없습니다. 대화 기록이 모두 삭제됩니다.
{% endhint %}

## 채팅방 설정 항목

### 자동 요약

* 대화가 길어지면 AI가 이전 내용을 자동으로 요약합니다.
* 자동 요약을 통해 대화의 맥락을 유지하면서도 원활한 대화가 가능합니다.

### 페르소나 변경

* 채팅방별로 다른 페르소나를 적용할 수 있습니다.
* 채팅방 설정에서 **페르소나**를 변경합니다.

### AI 모델 변경

* 채팅방별로 다른 AI 모델을 선택할 수 있습니다.
* 채팅방 설정에서 **AI 모델**을 변경합니다.

{% hint style="info" %}
채팅방 설정은 해당 채팅방에만 적용되며, 다른 채팅방에는 영향을 주지 않습니다.
{% endhint %}
