# 메모리북

{% hint style="info" %}
메모리북은 현재 베타 기능으로, 향후 업데이트에 따라 변경될 수 있습니다.
{% endhint %}

## 메모리북이란?

메모리북은 AI 캐릭터가 대화 내용을 기억할 수 있도록 도와주는 기능입니다. 메모리북을 사용하면 캐릭터가 이전 대화에서 언급된 중요한 정보를 기억하고, 이를 바탕으로 더 자연스러운 대화를 이어갈 수 있습니다.

## 주요 기능

### 자동 기억

* AI가 대화 중 중요한 정보를 자동으로 메모리북에 저장합니다.
* 사용자의 이름, 좋아하는 것, 싫어하는 것 등을 기억합니다.

### 수동 추가

* 사용자가 직접 메모리북에 정보를 추가할 수 있습니다.
* 채팅방 설정에서 메모리북을 열어 내용을 편집합니다.

### 메모리 삭제

* 잘못 기억된 정보나 불필요한 메모리는 삭제할 수 있습니다.
* 메모리북에서 해당 항목을 선택하여 삭제합니다.

## 메모리북 확인 방법

1. 채팅방에서 **설정** 아이콘을 클릭합니다.
2. **"메모리북"** 메뉴를 선택합니다.
3. 현재 저장된 메모리 목록을 확인할 수 있습니다.

## 활용 팁

* 장기적인 스토리라인이 있는 대화에서 특히 유용합니다.
* 캐릭터가 기억했으면 하는 중요한 설정은 수동으로 추가하는 것이 좋습니다.
* 메모리가 너무 많으면 캐릭터의 응답 품질이 떨어질 수 있으니, 핵심 정보 위주로 관리해 주세요.
