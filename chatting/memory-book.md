# 장기기억 & 기억 조각

## 장기기억이란?

장기기억은 AI가 작품 내 있었던 사건들을 **자동으로 기억**하는 기능입니다. 대화가 진행되면서 중요한 사건, 정보, 관계 등을 자동으로 저장하여 더 자연스러운 대화를 이어갑니다.

## 기억 조각이란?

기억 조각은 장기기억에서 캐치하지 못한 **세부 사건들을 자동으로 조각으로 기억**하는 기능입니다. 작은 디테일까지 놓치지 않고 대화에 반영됩니다.

## 메모리 확인 방법

1. 채팅방에서 **설정 메뉴(⋮)**를 클릭합니다.
2. **"메모리"**를 선택합니다.
3. 현재 저장된 기억 목록이 카드 형태로 표시됩니다.

### 메모리 검색

메모리 시트 상단의 검색창에서 키워드로 기억을 검색할 수 있습니다. 유사도 기반 벡터 검색으로 관련된 기억을 찾아줍니다.

### 메모리 사용 ON/OFF

설정 메뉴에서 **"메모리 사용"** 토글로 메모리 기능을 켜거나 끌 수 있습니다.

## 활용 팁

* 장기적인 스토리라인이 있는 대화에서 특히 유용합니다.
* 메모리 사용을 ON으로 유지하면 AI가 이전 대화 맥락을 더 잘 반영합니다.
* 대화가 길어져 AI가 이전 내용을 놓치는 경우, **대화 요약**과 **유저 노트**를 함께 활용하면 효과적입니다.

{% hint style="info" %}
장기기억과 기억 조각은 모두 자동으로 생성됩니다. 대화를 진행하면 AI가 알아서 중요한 사건을 기억합니다.
{% endhint %}
