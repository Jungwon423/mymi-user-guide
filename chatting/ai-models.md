# MYMI에서 사용할 수 있는 AI 모델 안내

## AI 모델이란?

MYMI에서는 여러 종류의 AI 모델을 제공합니다. 각 모델마다 특성이 다르며, 대화 스타일이나 목적에 맞게 선택하여 사용할 수 있습니다. 메시지 전송 시 **스파크**가 소비됩니다.

## 제공 모델

### Gemini 그룹

| 모델 | 특징 |
| --- | --- |
| **Gemini 3.1 Pro Preview** | 최신 프리뷰 모델 |
| **Gemini 3 Pro** | 높은 품질의 응답 |
| **Gemini Pro** | 안정적인 성능 |
| **Gemini 3 Flash** | 빠른 응답 속도 |
| **Gemini Flash** | 가장 빠른 응답 |

### Claude 그룹

| 모델 | 특징 |
| --- | --- |
| **Claude 4.6 Opus** | 최고 성능 모델 |
| **Claude 4.5 Opus** | 고성능 모델 |
| **Claude 4.6 Sonnet** | 성능과 속도의 균형 |
| **Claude 4.5 Sonnet** | 안정적인 성능 |
| **Claude 4 Sonnet** | 기본 성능 |
| **Claude** | 기본 모델 |

## 모델 변경 방법

1. 채팅방 하단 입력창 옆의 **모델 선택 버튼**을 클릭합니다.
2. 원하는 모델을 선택합니다.
3. 선택 즉시 적용됩니다.

## 모델 선택 팁

* **빠른 응답**이 필요하면 Flash 계열 모델을 선택하세요.
* **품질 높은 대화**가 필요하면 Opus나 Pro 계열 모델을 선택하세요.
* 모델을 변경해도 기존 대화 기록은 유지됩니다.
* 채팅방마다 다른 모델을 설정할 수 있습니다.

{% hint style="warning" %}
제공되는 모델의 종류와 사양은 업데이트에 따라 변경될 수 있습니다.
{% endhint %}
