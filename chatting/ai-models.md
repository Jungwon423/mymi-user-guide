# MYMI에서 사용할 수 있는 AI 모델 안내

## AI 모델이란?

MYMI에서는 여러 종류의 AI 모델을 제공합니다. 각 모델마다 특성이 다르며, 대화 스타일이나 목적에 맞게 선택하여 사용할 수 있습니다.

## 제공 모델

| 모델 | 특징 | 추천 용도 |
| --- | --- | --- |
| **기본 모델** | 빠른 응답 속도, 일상 대화에 적합 | 가벼운 대화, 일반적인 채팅 |
| **프로챗 모델** | 높은 품질의 응답, 복잡한 대화 가능 | 스토리 중심 대화, 깊이 있는 롤플레이 |

{% hint style="info" %}
프로챗 모드는 일반 채팅보다 훨씬 뛰어난 성능의 AI 모델을 사용해 채팅을 진행할 수 있는 모드입니다. 채팅창 좌측 하단의 토글로 ON/OFF가 가능합니다.
{% endhint %}

## 프로챗 획득 방법

프로챗은 다음 방법으로 획득할 수 있습니다:

* **서비스 내 구매**
* **출석 보상**
* **친구초대 보상**
* **제작자 보상**

## 모델 변경 방법

1. 채팅방에서 **설정** 아이콘을 클릭합니다.
2. **"AI 모델"** 메뉴를 선택합니다.
3. 원하는 모델을 선택합니다.

## 모델 선택 팁

* **빠른 응답**이 필요하면 기본 모델을 선택하세요.
* **품질 높은 대화**가 필요하면 프로챗 모드를 활성화하세요.
* 모델을 변경해도 기존 대화 기록은 유지됩니다.
* 채팅방마다 다른 모델을 설정할 수 있습니다.

{% hint style="warning" %}
제공되는 모델의 종류와 사양은 업데이트에 따라 변경될 수 있습니다.
{% endhint %}
