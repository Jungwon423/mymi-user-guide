# MYMI에서 사용할 수 있는 AI 모델 안내

## AI 모델이란?

MYMI에서는 여러 종류의 AI 모델을 제공합니다. 각 모델마다 특성이 다르며, 대화 스타일이나 목적에 맞게 선택하여 사용할 수 있습니다. 메시지 전송 시 **스파크**가 소비됩니다.

## 제공 모델

### Gemini 그룹

| 모델 | 특징 | 스파크 |
| --- | --- | --- |
| **Gemini 3.1 Pro** | 맥락 파악에 능한 차세대 프론티어 모델 | 100 |
| **Gemini 3 Flash** | 빠른 응답 속도와 우수한 성능의 균형 | 50 |
| **Gemini 3 Pro** | 효율적인 추론으로 뛰어난 표현력을 갖춘 최신 모델 | 100 |
| **Gemini 2.5 Pro** | 깊은 몰입감과 섬세한 표현력을 갖춘 인기 모델 | 80 |
| **Gemini 2.5 Flash** | 가성비 좋은 빠른 응답의 실속 모델 | 40 |

### Claude 그룹

| 모델 | 특징 | 스파크 |
| --- | --- | --- |
| **Claude Sonnet 4.6** | 감정 표현이 섬세하고 대화에 깊이가 있는 최신 모델 | 100 |
| **Claude Opus 4.6** | 최정점 추론 지능의 프론티어 모델 | 300 |
| **Claude Opus 4.5** | 최고의 추론 능력과 창의적 글쓰기의 정점 | 300 |
| **Claude 4.5 Sonnet** | 빠르고 안정적인 성능의 올라운더 AI | 100 |
| **Claude 4 Sonnet** | 자연스러운 대화와 정확한 정보 처리 | 100 |
| **Claude 3.7 Sonnet** | 감성적이고 공감 능력이 뛰어난 대화 모델 | 100 |

## 모델 변경 방법

1. 채팅방 하단 입력창 옆의 **모델 선택 버튼**을 클릭합니다.
2. 원하는 모델을 선택합니다.
3. 선택 즉시 적용됩니다.

## 모델 선택 팁

* **빠른 응답**이 필요하면 Flash 계열 모델을 선택하세요.
* **품질 높은 대화**가 필요하면 Opus나 Pro 계열 모델을 선택하세요.
* 모델을 변경해도 기존 대화 기록은 유지됩니다.
* 채팅방마다 다른 모델을 설정할 수 있습니다.

{% hint style="warning" %}
제공되는 모델의 종류와 사양은 업데이트에 따라 변경될 수 있습니다.
{% endhint %}
