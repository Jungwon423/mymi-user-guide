# 캐릭터와 대화하기

## 캐릭터 찾기

### 탐색

1. 메인 페이지에서 다양한 캐릭터를 탐색할 수 있습니다.
2. **카테고리별**, **인기순**, **최신순** 등으로 필터링이 가능합니다.
3. 원하는 캐릭터를 클릭하면 캐릭터 상세 페이지로 이동합니다.

### 검색

1. 상단의 **검색** 기능을 사용합니다.
2. 캐릭터 이름, 태그, 설명 등으로 검색할 수 있습니다.

## 대화 시작하기

1. 캐릭터 상세 페이지에서 **"대화하기"** 버튼을 클릭합니다.
2. 새로운 채팅방이 생성되며 캐릭터의 첫 인사가 표시됩니다.
3. 하단 입력창에 메시지를 입력하고 전송합니다.

## 대화 중 기능

### 메시지 재생성

캐릭터의 답변이 마음에 들지 않으면 **재생성** 버튼을 클릭하여 새로운 답변을 받을 수 있습니다.

### 메시지 편집

내가 보낸 메시지를 수정하고 싶다면 해당 메시지를 길게 눌러 **편집** 기능을 사용할 수 있습니다.

### 메시지 삭제

불필요한 메시지는 삭제하여 대화 흐름을 정리할 수 있습니다.

## 세이프티 필터

MYMI는 기본적으로 세이프티 필터가 ON 상태입니다.

* **세이프티 필터 ON**: 안전한 대화만 가능합니다.
* **세이프티 필터 OFF**: 성인인증 완료 후 사용 가능합니다.

{% hint style="info" %}
세이프티 필터 설정은 [마이] → [세이프티 필터]에서 변경할 수 있습니다.
{% endhint %}
