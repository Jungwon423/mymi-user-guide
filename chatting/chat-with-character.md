# 대화하기

## 작품 찾기

### 탐색

메인 페이지에서 다양한 방법으로 작품을 탐색할 수 있습니다:

* **추천**: 개인화된 추천 작품
* **신작**: 새로 출시된 작품
* **랭킹**: 실시간/일간/주간/월간 인기 순위
* **태그**: 장르, 관계, 성향, 종족 등 태그별 탐색

### 필터

* **성별 필터**: 전체 / 남성 / 여성 작품 필터링
* **세이프티 필터**: 민감한 콘텐츠 표시 여부 (성인인증 필요)

### 검색

상단의 **검색** 탭에서 작품 이름, 태그, 설명 등으로 검색할 수 있습니다.

## 대화 시작하기

1. 원하는 작품을 클릭하여 상세 페이지로 이동합니다.
2. **"대화하기"** 버튼을 클릭합니다.
3. 새로운 채팅방이 생성되며 첫 인사가 표시됩니다.
4. 하단 입력창에 메시지를 입력하고 전송합니다.

## 채팅 모드

3가지 모드를 헤더의 **UI 설정 아이콘**에서 전환할 수 있습니다.

### 채팅 모드 (Chat)

기본 모드입니다. 말풍선 형식으로 메시지가 표시됩니다.

### 1:1 모드 (One-to-One)

비주얼 노벨 스타일로, 메시지가 하나씩 중앙에 크게 표시됩니다. 좌우 화살표로 이전/다음 메시지를 탐색할 수 있습니다.

### 노벨 모드 (Novel)

소설 읽기 형식으로, 서술형 텍스트와 이미지가 통합 렌더링됩니다. 책 읽는 경험에 가장 가깝습니다.

{% hint style="info" %}
처음 채팅에 진입하면 3가지 모드를 안내하는 가이드가 표시됩니다.
{% endhint %}

## 대화 중 기능

### 메시지 재생성

응답이 마음에 들지 않으면 **재생성** 버튼을 클릭하여 새로운 답변을 받을 수 있습니다. 기존 답변은 후보 메시지로 보존되며, 좌우 화살표로 후보 간 전환이 가능합니다.

### 메시지 편집

메시지에 호버하면 나타나는 **편집** 버튼으로 내용을 수정할 수 있습니다.

### 메시지 삭제

불필요한 메시지는 **삭제** 버튼을 클릭하여 제거할 수 있습니다. 확인 모달이 표시됩니다.

## 세이프티 필터

MYMI는 기본적으로 세이프티 필터가 ON 상태입니다.

* **세이프티 필터 ON**: 안전한 대화만 가능합니다.
* **세이프티 필터 OFF**: 성인인증 완료 후 사용 가능합니다.

{% hint style="info" %}
세이프티 필터는 홈 화면 또는 [마이] → 민감한 콘텐츠 표시에서 변경할 수 있습니다.
{% endhint %}
